## Question

When should I use SQL vs NoSQL databases?

### Choose SQL When:
- Data is structured and relational
- Complex queries needed
- ACID compliance critical
- Schema is stable

### Choose NoSQL When:
- Unstructured data
- High write throughput
- Flexible schema needed
- Horizontal scaling priority
